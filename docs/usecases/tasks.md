## Задачи

### Создание задачи

Форма создания открывается из рабочего пространства

Форма создания задачи:
- Название (name)
- Описание (description)

Эндпоинты:
- `POST /tasks`: Request {workspace_id, name, description} Response {id, name, description, is_active, *other}

Операция разрешена только для пользователей с ролями Преподаватель и Владалец в пространстве


### Активация/деактивация задачи

В интерфейсе выглядит как тумблер, который перключает задачу из активного состояния в неактивное

Решения могут приниматься только для активных задач

Эндпоинты:
- `PATCH /tasks/:id/active`: Request {is_active} Response {id, name, description, is_active, *other}

Операция разрешена только для пользователей с ролями Преподаватель и Владалец в пространстве


### Редактирование задачи

Форма редактирования содержит все поля


Эндпоинты:
- `PUT /tasks/:id`: Request {name, description, is_active, *other}, Response {id, name, description, is_active, *other}

Операция разрешена только для пользователей с ролями Преподаватель и Владалец в пространстве


### Получение информации о задачи (только для преподавателей)

- `GET /tasks/:id`: Response {id, name, description, *other} 

Операция разрешена только для пользователей с ролями Преподаватель и Владалец в пространстве


### Получение информации о задачи (для всех)

- `GET /tasks/:id/public`: Response {id, name, description} 

Операция разрешена любым участникам пространства

Задача должна быть активной
