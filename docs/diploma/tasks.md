## Задание на ВКР

### Основные вопросы, подлежащие разработке

1. Провести анализ подходов по автоматизированной оценке работ на курсах по направлению программной инженерии
2. Спроектировать систему автоматизированной оценки на базе технологий Искусственного Интеллекта
3. Разработать прототип системы автоматизированной оценки
4. Провести тестирование прототипа на выделенной группе студентов
5. На основе обратной связи подготовить перечень улучшений для доработки целевой системы


### Форма представления материалов

Для представления результатов ВКР будет предоставлен текст работы
